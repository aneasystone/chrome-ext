Page Action动画图标
--------------------
使用 background_page, page_action and tabs
这个扩展在工具栏上添加一个随浏览而变动的Page Action图标

##调用:
 - chrome.pageAction.hide
 - chrome.pageAction.onClicked
 - chrome.pageAction.setIcon
 - chrome.pageAction.setTitle
 - chrome.pageAction.show
 - chrome.tabs.get

##源文件:
 - background.html
 - manifest.json
