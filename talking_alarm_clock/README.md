会说话的闹钟
---------------
使用 background, background_page, browser_action and tts
有两种警报，可以发出一段声音并且说出一条被选择的短句

##调用:
 - chrome.browserAction.setIcon
 - chrome.extension.connect
 - chrome.extension.onConnect
 - chrome.tts.getVoices
 - chrome.tts.speak
 - chrome.tts.stop
