新闻阅读(by Google)
-----------------
使用 background_page, browser_action, options_page and tabs
在弹窗中显示Google新闻中最新的Story

##调用:
 - chrome.extension.getURL
 - chrome.i18n.getMessage
 - chrome.tabs.create

 - _locales/en/messages.json

 - css/options.css
 - javascript/feed.js
 - javascript/options.js
 - javascript/util.js
 - manifest.json
 - views/background.html
 - views/feed.html
 - views/options.html
