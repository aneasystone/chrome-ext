Imageinfo
-------------
使用 background_page, contextMenus and tabs
从Images中获取图片信息,包含 EXIF 数据

##调用:
 - chrome.contextMenus.create
 - chrome.tabs.get
 - chrome.tabs.getCurrent
 - chrome.windows.create
 - chrome.windows.update

 - imageinfo/exif.js
